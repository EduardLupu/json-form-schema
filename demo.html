Form Builder Component Demo

A powerful, customizable form builder web component that generates forms from JSON schemas.

Customizable Fields

Support for text, email, password, number, textarea, select, checkbox, and radio inputs with extensive styling options.

Searchable Select

Enhanced select fields with search functionality for better user experience with large option sets.

Custom Styling

Fully customizable styles for each field, including colors, sizes, and custom CSS classes.

Responsive Design

Built-in responsive design that works great on all devices and screen sizes.

Validation

Built-in form validation with customizable error messages and visual feedback.

Method 1: JavaScript Schema

Set the schema programmatically using JavaScript:

const formBuilder = document.querySelector('#form1');
const schema = {
  fields: [
    {
      type: 'text',
      name: 'username',
      label: 'Username',
      required: true,
      placeholder: 'Enter your username',
      style: {
        width: '100%',
        backgroundColor: '#f8f9fa',
        borderRadius: '8px'
      }
    },
    {
      type: 'email',
      name: 'email',
      label: 'Email Address',
      required: true,
      placeholder: 'Enter your email',
      helpText: 'We'll never share your email with anyone else.'
    },
    {
      type: 'select',
      name: 'country',
      label: 'Country',
      required: true,
      searchable: true,
      options: [
        'United States',
        'Canada',
        'United Kingdom',
        'Australia',
        'Germany',
        'France',
        'Japan',
        'China',
        'India',
        'Brazil'
      ]
    },
    {
      type: 'number',
      name: 'age',
      label: 'Age',
      min: 18,
      max: 120,
      step: 1,
      style: {
        width: '150px'
      }
    }
  ]
};
formBuilder.setAttribute('schema', JSON.stringify(schema));


Method 2: HTML Attribute Schema

Define the schema directly in the HTML using the schema attribute:

<form-builder
  submit-text="Submit"
  schema='[
    {
      "type": "text",
      "name": "fullName",
      "label": "Full Name",
      "required": true,
      "style": {
        "width": "100%",
        "backgroundColor": "#f8f9fa"
      }
    },
    {
      "type": "select",
      "name": "favoriteColor",
      "label": "Favorite Color",
      "searchable": true,
      "options": [
        "Red",
        "Blue",
        "Green",
        "Yellow",
        "Purple",
        "Orange",
        "Pink",
        "Black",
        "White"
      ]
    },
    {
      "type": "checkbox",
      "name": "terms",
      "label": "I agree to the terms and conditions"
    }
  ]'
></form-builder>
